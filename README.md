## Atividades- Curso Javascript - Variaveis e tipos.

Atividades praticas propostas no curso "Javascript - Variáveis e Tipos" da Digital Innovation One, ministrado por Stephany Nusch.

##### Atividade 1

Verificar de duas maneiras diferentes entre si se uma String é um palíndromo.
Palíndromo: frase ou palavra que se pode ler, indiferentemente da esquerda para direita ou vice-versa.
(Ex: ralar, ovo, ana, radar)

##### Atividade 2

Troque todos os elementos pares e diferentes de 0, pelo número 0, e se o array for vazio, retorne -1.
